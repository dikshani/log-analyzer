# Log Analyzer

Shell script to analyze Nginx access logs.

## Features
- Top 5 IP addresses
- Top 5 requested paths
- Top 5 response status codes
- Top 5 user agents

## Run

```bash
chmod +x analyze_logs.sh
./analyze_logs.sh nginx-access.log
```
